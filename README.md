# 🛒 Product Management System (MERN Stack)

A full-stack Product Management Web Application built using the MERN Stack (MongoDB, Express, React, Node.js).

This application allows users to register, login, and manage their own products with category support.

# 🚀 Features

5️⃣ Run Frontend
